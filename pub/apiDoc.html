How to Create Time Archive Objects

The Time Archive library has 3 types of objects that can be created.
Each object corresponds to a type of timeline that a user can make.
Each type of Time Archive object requires a location parameter for their constructor.
The location parameter refers to where the user wishes to add the newly made timeline.
If a location is not specified then the newly created Time Archive object is added to the body of an HTML file.

The 'PopUpTimeline' Object

In addition to a location parameter, the 'PopUpTimeline' also has a timeline style parameter. The timeline style parameter indicates which type of 'PopUpTimeline' style the user desires. The possible 'PopUpTimeline' styles are regular and soft. The regular style has rectangular popups that can
be previewed if a user hovers over the popup's respective date on the timeline. The regular style also allows the user to close a selected popup via
an 'X' located on the selected popup. The soft style doesn't have the preview option that the regular style has, but it has buttons on the timeline that
can be clicked to open a popup. When a button's date is clicked, the button's color changes, indicating that its been selected and the respective
popup opens up. Unlike the regular style popup, the soft style popups can be closed by clicking away from the activated popup. If a timeline style is not specified in the 'PopUpTimeline' constructor then the timeline style is set to 'regular'.

A 'PopUpTimeline' objected can be created like this: 

const historyofChocolate = new PopUpTimeline("#pop", "regular");

The 'SlideShowTimeLine' Object

Like the 'PopUpTimeline', the 'SlideShowTimeLine' also takes in a style parameter upon initialization. The possible timeline styles for the 'SlideShowTimeLine' are 'gallery' and 'regular'. The 'gallery' style has a slide for the timeline's title. The slide navigation buttons for this style are also on the left and right of the slide display. Unlike the 'gallery' style, the 'regular' style displays the timeline's title above the slide view. Also, the 'regular' style has the slide navigation buttons below the slide view. If a timeline style is not specified in the 'SlideShowTimeLine' constructor then the timeline style is set to 'regular'.

A 'SlideShowTimeLine' objected can be created like this: 

const katyPerryDiscography = new SlideShowTimeLine("#slide", "gallery");

The 'StandardTimeLine' Object

Unlike the 'PopUpTimeline' and 'SlideShowTimeLine' objects, the 'StandardTimeLine' doesn't have a timeline style parameter.
It only has a location parameter.

A 'StandardTimeLine' objected can be created like this: 

const Castlevania = new StandardTimeLine("#standardEx");

    deletePopup index:Int This method deletes the popup at the 'index' argument provided. The popup is deleted from the rendered timeline and the list keeping track of the timeline's popups. If the index is invalid (out of bounds), then this method doesn't alter the list keeping track of all the popups. If no argument is provided, then the method doesn't alter anything.
    sortDecreasingPopups N/A This method sorts the popups by their date property. The popups are sorted in decreasing order and the timeline is re-rendered with the newly sorted popups.
    sortIncreasingPopups N/A This method sorts the popups by their date property. The popups are sorted in increasing order and the timeline is re-rendered with the newly sorted popups.
